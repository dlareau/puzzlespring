# Creating Hunts

This section covers everything you need to create and configure a puzzle hunt in PuzzleSpring, from basic hunt setup to advanced puzzle unlocking rules.

## Who Is This For?

This section is for **hunt organizers** who need to:

- Create new puzzle hunts and puzzles
- Configure puzzle content and files
- Set up puzzle unlocking rules
- Configure hints and answer validation
- Create pre-puzzles for public access

## Recommended Workflow

For your first hunt, we recommend following these steps in order:

1. **Create the hunt and puzzles** - Set up the basic structure
2. **Add puzzle content** - Upload files and configure templates
3. **Configure unlock rules** - Define how puzzles become available
4. **Set up hints** - Configure hint allocation and policies
5. **Configure answer responses** - Set up custom response messages
6. **Create prepuzzles** - Optional pre-hunt puzzles for public access
7. **Test thoroughly** - Use the Config Tester to verify your setup

# Installation

This section covers everything you need to get PuzzleSpring up and running, from a quick Docker setup to production deployment and server configuration.

## Who Is This For?

This section is for **server administrators** who need to:

- Deploy PuzzleSpring on a server
- Configure environment variables and application settings
- Set up SSL certificates and domain names
- Manage database and service infrastructure

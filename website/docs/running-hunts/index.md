# Running Hunts

This section covers the tools and processes for managing a live puzzle hunt, from monitoring team progress to responding to hints.

## Who Is This For?

This section is for **hunt staff** who need to:

- Monitor team progress during a hunt
- Respond to hint requests
- Manage teams and participants
- Send announcements and updates
- Import or export hunt data

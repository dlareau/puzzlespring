Build and host custom puzzlehunts with ease

PuzzleSpring is a Django-based puzzlehunt hosting platform designed to be easy to set up and customize while maintaining powerful features for both hunters and hunt operators.

Flexible Puzzle Structure

Complete flexibility in hunt structure, with customizable puzzle unlocking rules.

Easy Customization

Multiple customization options, from no-code configuration to full template control.

Hint System

Built-in hint request and response system with customizable hint unlocking.

Team Management

Team registration and management with configurable size limits and real-time progress tracking.

Smart Notifications

Keep teams engaged with configurable notifications via Discord, email, and more.

Real-Time Updates

Instant synchronization of submissions, hints, and unlocks across all team members.

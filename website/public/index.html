Build and Host Custom Puzzlehunts with Ease

PuzzleSpring is a Django-based puzzlehunt hosting platform designed to be easy to set up and customize while maintaining powerful features for both hunters and hunt operators.

Flexible Puzzle Structure

Support for linear, nonlinear, and metapuzzle structures with customizable unlocking conditions.

Team Management

Comprehensive team registration and management system with size limits and waiting lists.

Hunt Scheduling

Schedule hunts in advance with automatic starting and stopping functionality.

Hint System

Built-in hint request and response system with customizable limitations.

Statistics & Analytics

Comprehensive statistics tracking and visualization for teams and puzzles.

Easy Customization

Highly customizable with straightforward Django templates and styling options.
